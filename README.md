# 💲 Finanztracker 

Ein einfaches Finanzverwaltungssystem zur Erfassung und Verwaltung von Einnahmen und Ausgaben.

## 💸 Projektbeschreibung

Im Rahmen eines Schulprojekts entwickeln wir eine Desktop-Anwendung zur Verwaltung persönlicher Finanzen.  
Das Programm ermöglicht es Benutzern, Einnahmen und Ausgaben zu erfassen, zu speichern und auszuwerten.

Der Fokus liegt auf einer sauberen Programmlogik, strukturierter Datenverwaltung und einer übersichtlichen Anwendung.

## 👾 Funktionen (geplant)

- Einnahmen hinzufügen
- Ausgaben hinzufügen
- Transaktionen anzeigen
- Gesamtsaldo berechnen
- Speicherung der Daten in einer Datenbank

Optional (Wenn Zeit bleibt):
- JavaFX Benutzeroberfläche
- Statistische Auswertung

## 🌪️ Technologien

- Java
- Maven
- <b>mögliche</b> GUI mit JavaFX
- Datenbank (MySQL)
- Git & GitHub
